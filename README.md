# taptik-labs

Agent global settings for OpenCode. Shared workflow, memory, and canon configuration.

## What This Repo Contains

- `AGENTS.md` - shared agent operating rules
- `MEMORY.md` - startup memory index for durable behavior notes
- `skills/` - reusable skills, references, templates, and helpers

## Folder Overview

- `skills/browser/` - browser automation skill docs
- `skills/canon/` - canon task-note workflow for complex work
- `skills/gh/` - GitHub CLI skill docs
- `skills/memories/` - memory-note workflow and search helpers

## Role

This repository acts as the global agent configuration source for OpenCode sessions.
